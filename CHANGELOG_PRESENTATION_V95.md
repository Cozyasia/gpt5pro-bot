# Changelog — Presentation Studio v95

## Базовая версия

`main - 2026-07-14T134403.875.py`

## Изменения в main.py

- Версия изменена на `v95-presentation-studio-fix-2026-07-14`.
- Размер автоматического пакета изображений по умолчанию увеличен до 10.
- Добавлен `PRESENTATION_IMAGE_SIZE=1536x1024`.
- Усилен системный промпт мастера структуры и контроля фактов.
- Добавлена отдельная горизонтальная генерация изображений для презентаций.
- Автоматический режим использует стабильную одиночную генерацию; Midjourney остаётся ручным выбором.
- Сбой отдельного изображения больше не сдвигает соответствие «изображение → слайд».
- Для callback `ps:*` добавлена локальная обработка ошибок.
- Глобальный error handler не отправляет второе ложное сообщение об ошибке внутри мастера.

## Новый presentation_studio.py

- Новый устойчивый конечный автомат мастера.
- Отдельная таблица `presentation_projects_v95` для безопасной миграции без конфликта со старой схемой.
- Точное извлечение названия бренда, слогана, продуктов, цен и контактов.
- Накопительные уточнения логотипа, изображений, стиля и палитры.
- Четыре сценария логотипа.
- Точное соблюдение количества слайдов.
- Единая модель контента для PDF и PPTX.
- Сохранение состояния после каждого этапа.
